Live embed-config propagation

This page mounts the local chat-embed.js with an embedId set so it opens the WS listener at /ws/embed/<id>/. Edit the embed config in the dashboard and watch the launcher / theme update without a page reload.

1. Set the embedId

Replace REPLACE_ME below with an emb_… id from the dashboard (Agent → Embed → Generate embed code).

2. Open DevTools → Network → WS

You should see ws://localhost:8000/ws/embed/<id>/ connect. When you save in the dashboard, an inbound frame {"type":"embed_config_updated", ...} arrives.

3. Verify the change

Change the primary colour in the dashboard → Save → the launcher bubble in the bottom-right flips colour within ~1 s.
